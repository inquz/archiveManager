Работа с архивом жильцов

Заголовочный файл содержит функции для загрузки, сохранения, отображения и управления архивом жильцов, организованным в виде стека.

◆ clearArchive()

void clearArchive ( Stack * stack)

Очищает архив

Удаляет все элементы из стека и освобождает занимаемую память. После выполнения стек остается пустым и готовым к повторному использованию.

Аргументы
stackУказатель на стек для очистки

◆ loadDataFromInput()

void loadDataFromInput ( Stack * stack,
const char * filename )

Загружает данные из файла формата input.txt.

Читает данные из текстового файла в формате input.txt и помещает их в стек. Формат файла должен содержать полные данные о жильцах.

Аргументы
stackУказатель на стек для заполнения
filenameИмя файла для чтения (input.txt)
Заметки
Формат файла input.txt: Номер_карты Адрес ФИО Кол-во_жильцов Дата_заселения Общая_площадь Избыточная_площадь
