◆ addTenantToInput()

void addTenantToInput ( Stack * stack)

Добавляет нового жильца в формате input.txt.

Интерактивно запрашивает у пользователя все необходимые данные для полной записи о жильце и добавляет их в стек. Включает все поля структуры Tenant.

Аргументы
stackУказатель на стек, куда будет добавлена новая запись
Заметки
Запрашиваемые данные:
  • Номер карточки
  • Адрес проживания
  • ФИО жильца
  • Количество проживающих
  • Дата заселения (в формате DD.MM.YYYY)
  • Общая площадь помещения
  • Избыточная площадь
Предупреждения
Проверяет корректность вводимых данных (особенно числовых значений)

◆ addTenantToKodif()

void addTenantToKodif ( Stack * stack)

Добавляет нового жильца в формате kodif.txt.

Интерактивно запрашивает у пользователя ограниченный набор данных для сокращенной записи о жильце и добавляет их в стек.

Аргументы
stackУказатель на стек, куда будет добавлена новая запись
Заметки
Запрашиваемые данные:
  • Номер карточки
  • Адрес проживания
  • ФИО жильца
  • Количество проживающих
Предупреждения
Проверяет уникальность номера карточки в архиве
